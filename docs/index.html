Início

Bem-vindo ao Pitchutcha

Uma jornada profunda e detalhada através das eras da computação, desvendando as mentes, as máquinas e as ideias que moldaram o nosso mundo digital.

O Projeto Pitchutcha nasceu da ambição de criar um recurso centralizado, aberto e extremamente detalhado sobre a ciência da computação. Nossa missão é documentar não apenas o "como", mas o "porquê" e o "quem" por trás de cada avanço tecnológico, desde os conceitos matemáticos mais abstratos até as aplicações práticas que usamos todos os dias.

Navegue pelo Conhecimento: Utilize o menu lateral para explorar as diferentes eras e disciplinas da computação. A busca no topo da página permite encontrar tópicos específicos rapidamente.

Sobre o Projeto Pitchutcha

Pitchutcha é mais do que um site de documentação; é um esforço colaborativo para construir a enciclopédia de ciência da computação mais abrangente e acessível do mundo. Acreditamos que o conhecimento sobre as ferramentas que moldam nosso mundo deve ser livre, detalhado e compreensível para todos, desde estudantes curiosos até profissionais experientes.

Nossos princípios fundamentais são:

  • Profundidade e Precisão: Vamos além do superficial, conectando teorias, pessoas e eventos para criar uma narrativa coesa e precisa.
  • Código Aberto e Colaborativo: Todo o conteúdo e o código do site são abertos. Acreditamos no poder da comunidade para criar um recurso melhor para todos.
  • Acessibilidade Universal: O design e a estrutura são pensados desde o início para serem acessíveis a pessoas com diferentes habilidades e necessidades, seguindo as diretrizes do WCAG.
  • Design Focado na Leitura: A experiência do usuário é otimizada para longas sessões de leitura, com tipografia clara, temas adaptáveis e uma interface livre de distrações.

Como Contribuir

Este projeto é construído pela comunidade, para a comunidade. Existem muitas maneiras de ajudar, independentemente do seu nível de habilidade técnica:

1. Reportando Erros e Sugerindo Melhorias

Encontrou um erro de digitação, uma informação imprecisa ou tem uma ideia para um novo tópico? Abra uma "Issue" em nosso repositório no GitHub. Esta é uma das formas mais valiosas de contribuição.

2. Escrevendo e Editando Conteúdo

Se você tem conhecimento em uma área específica, pode contribuir diretamente com o conteúdo. O processo é feito através de "Pull Requests" no GitHub. Consulte nosso Guia de Contribuição para aprender o passo a passo.

3. Ajudando com o Código

Se você é um desenvolvedor, pode ajudar a melhorar o próprio site, seja corrigindo bugs, melhorando a acessibilidade, otimizando o CSS ou adicionando novas funcionalidades ao JavaScript. Explore as issues para iniciantes.
